Sitio web para BsideFactory — creado con React y Tailwind CSS.

Descripción corta:

- Sitio web frontend construido con React (TypeScript) y Vite. Estilado con Tailwind CSS. Integración con Supabase.

Cómo ejecutar (desarrollo):

1. Instala dependencias:

```powershell
npm install
```

2. Ejecuta el servidor de desarrollo:

```powershell
npm run dev
```

Scripts útiles (en `package.json`):

- `dev` — inicia Vite en modo desarrollo
- `build` — construye para producción
- `preview` — sirve la build en modo preview
- `typecheck` — ejecuta `tsc --noEmit`
